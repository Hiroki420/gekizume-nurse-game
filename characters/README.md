# キャラクター画像の置き方

このフォルダに、看護師のキャラクター画像を配置してください。

## ファイル名の規則

- **4月の先輩（圧）**: `senpai_04_表情名.png`
  - `senpai_04_normal.png` - 通常
  - `senpai_04_cold.png` - 冷たい
  - `senpai_04_rage.png` - 激詰め
  - `senpai_04_smile.png` - 笑顔（任意）

## 画像の推奨サイズ

- 幅: 200-400px 程度
- 高さ: 300-600px 程度（全身 or 上半身）
- 形式: PNG（透明背景推奨）

## 画像がない場合

画像がなくてもゲームは動作します。その場合は、シルエットの枠だけが表示されます。

## 画像を直接貼り付ける場合

もし画像ファイルを直接ここに貼り付けたい場合は、以下の方法があります：

1. **画像をドラッグ&ドロップ**でこのフォルダに配置
2. または、画像をBase64エンコードしてコードに埋め込む（要相談）
