# 背景画像の配置方法

このフォルダに背景画像を配置してください。

## ファイル名の規則

背景画像は以下のファイル名で配置してください：

- `nurse_station.png` - ナースステーション
- `hallway.png` - 廊下
- `patient_room.png` - 病室
- `break_room.png` - 休憩室

## 画像の推奨サイズ

- 幅: 1920px 程度（フルHD推奨）
- 高さ: 1080px 程度
- 形式: PNG または JPG

## 画像がない場合

背景画像が配置されていない場合、デフォルトのグラデーション背景が表示されます。
